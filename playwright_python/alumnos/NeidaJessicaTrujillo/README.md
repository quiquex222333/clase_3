# Playwright con Python Jessica

Como usar

crear un entorno virtual

```
python -m venv venv
```

moverse al entorno
```
.\venv\Scripts\Activate.ps1 *windows
```


instalaciones minimas
```
pip install pytest-playwright
playwright install
pytest-html
```

Como ejecutar pruebas
```
pytest
pytest -v
pytest -x
pytest --html=pytest_report.html

```
